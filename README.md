## ABOUT

This is a simple calculator Java Swing app. Just in medieval theme.

## FEATURES

### OPERATIONS

- Addition
- Subtraction
- Multiplication
- Division
- Exponentiation

### CONSTANTS

- pi
- tau
- e
- golden ratio (phi)

### FUNCTIONS

- sqrt = Square Root
- abs = Absolute Value
- sin = Sinus
- cos = Cosinus
- tan = Tangent
- cot = Cotangent
- rad = Takes degrees and converts them to radians
- deg = same as 'rad' but the other way around
- round = Round
- floor = Floor
- ceil = Ceil
